# Архитектура

## Выбор стека

- **.NET 8 + Avalonia UI** — современный интерфейс, Windows 10/11 сейчас и путь к
  macOS без переписывания экранов и бизнес-логики.
- **Windows platform adapter** — низкоуровневый keyboard hook, `SendInput`,
  управление раскладкой, активным окном и безопасными полями вынесены из ядра.
- **Whisper.net 1.9 + NAudio** — офлайн-распознавание без ключей и запись микрофона.
- **JSON settings** в профиле пользователя; секреты будущих облачных провайдеров —
  только через системное защищённое хранилище.

## Модули

```text
FoturTypingHelper.App       интерфейс, tray, overlay, composition root
FoturTypingHelper.Core      правила раскладки, словари, настройки, статистика
FoturTypingHelper.Windows   hooks, активное окно, SendInput, автозапуск, аудио
FoturTypingHelper.Tests     модульные тесты чистой логики
installer                   Inno Setup и выпуск
```

Core не вызывает Win32 и остаётся переносимым. Системные события проходят через
последовательную очередь, чтобы hook никогда не блокировал ввод. Замена выполняется
транзакцией: сохранить активное окно → удалить ошибочный фрагмент → вставить
исправление → восстановить/сменить раскладку → записать обратимую операцию.

## Ограничения ОС

Windows запрещает процессу обычного уровня надёжно управлять окнами более высокого
уровня целостности. Постоянный запуск всего помощника от администратора ухудшает
безопасность. Предпочтительный будущий вариант — подписанный UIAccess-компонент;
до сертификата пользователь может по необходимости запустить приложение с тем же
уровнем прав, что и целевое приложение.
