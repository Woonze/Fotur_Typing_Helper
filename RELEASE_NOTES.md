# Fotur Typing Helper 1.0.0

Первый кроссплатформенный open-source релиз Fotur Typing Helper для Windows x64, macOS Apple Silicon и macOS Intel.

## Главное

- Автокоррекция теперь оценивает слова внутри всей фразы, восстанавливает смешанные фрагменты и сохраняет корректные имена, регистр и `Fotur`.
- Автоматический Chrome-стенд вырос до 150 RU/EN-фраз; перед каждым вводом подтверждается противоположная раскладка.
- Диктовка получила выбор микрофона, живой уровень, Small по умолчанию, VAD, noise gate, словарный prompt, обработку пунктуации и слов-паразитов.
- Обычное RU/EN-распознавание и локальный перевод на английский остаются отдельными режимами.
- Вместо микрофонного виджета запись показывает пульсирующая рамка фирменных цветов по краям всех мониторов.
- Добавлена автоматическая проверка стабильных GitHub Releases при запуске и установка пакета после сверки SHA-256.
- Экран «О программе» показывает создателя Кирилла/Woonze, версию, дату, платформу, лицензию и репозиторий.

## macOS

Добавлены нативные CGEvent/TIS-клавиатура и вставка, OpenAL microphone capture, Whisper CoreML/CPU, LaunchAgent, tray и отдельные DMG/ZIP для arm64 и x64. GitHub Actions собирает архитектуры на раздельных ARM/Intel runner и проверяет bundle, подпись и Whisper dylib.

## Важно перед установкой

- Модель загружается при первом использовании; Small занимает около 466 МБ.
- macOS запросит Microphone, Accessibility и Input Monitoring.
- Текущие Windows/macOS пакеты не подписаны коммерческими сертификатами. macOS DMG имеет ad-hoc подпись, но не notarization; Gatekeeper может потребовать «Всё равно открыть».
- Потоковый предварительный текст и перевод не на английский ещё находятся в backlog. Финальная вставка после завершения записи работает локально.
