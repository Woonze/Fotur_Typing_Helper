# Fotur Typing Helper 0.2.0

Большое обновление интерфейса и основных функций Windows-версии.

## Что нового

- Полностью переработан интерфейс: единая тёмная тема Fotur, понятные статусы, описания функций и `i`-подсказки.
- Автокоррекция теперь анализирует контекст и может заменить неверно набранный непрерывный фрагмент длиной до 24 слов.
- Добавлено экспериментальное раннее исправление слова во время набора.
- Горячие клавиши диктовки и отмены можно переназначать; конфликты и системные сочетания проверяются до сохранения.
- Подготовлены Setup, portable ZIP и SHA-256 для проверки скачанных файлов.

## Исправлено

- Устранена ошибка загрузки Windows native runtime локальной Whisper-диктовки: DLL теперь поставляются отдельными файлами и проверяются сборочным скриптом.
- Исправлено x64-описание WinAPI `INPUT`: раньше Windows отклоняла события Backspace и вставки, поэтому раскладка переключалась, но предыдущий текст не заменялся.
- Перед диктовкой запоминается активное окно, а перед вставкой текста ему возвращается фокус.
- Снижены задержки глобального перехватчика клавиатуры: активное окно кэшируется, зависающие Win32-вызовы ограничены по времени.
- Исправление раскладки учитывает предыдущие слова, а не только последнее завершённое слово.

## Важно

- Это предварительный open-source выпуск для Windows x64. Перед стабильным релизом продолжается ручная проверка в разных редакторах и на разных микрофонах.
- Локальная модель диктовки загружается при первом использовании. Крупные модели точнее, но работают медленнее и требуют больше памяти.
- Установщик пока не подписан сертификатом и может вызвать предупреждение Microsoft Defender SmartScreen.
