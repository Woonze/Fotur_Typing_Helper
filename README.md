# Fotur Typing Helper

Локальный помощник ввода для Windows 10/11. Он исправляет русские и английские слова и фразы, набранные не в той раскладке, переключает раскладку для продолжения печати и вставляет голосовую диктовку в активное поле.

Версия 0.3.0 разделяет обычную двуязычную диктовку и локальный перевод речи на английский, надёжнее исправляет целые фразы с заглавными буквами и клавишами пунктуации, а также устраняет залипание режима «удерживать, пока говорю». Навигация получила единый набор векторных иконок, а экран «О программе» — подробное описание продукта и приватности.

Приложение ориентировано на приватность: анализ клавиатуры и базовая диктовка работают локально, содержимое текста не записывается в журнал и не отправляется по сети. Архитектура допускает будущие версии для macOS и необязательные облачные BYOK-провайдеры.

Репозиторий: [Woonze/Fotur_Typing_Helper](https://github.com/Woonze/Fotur_Typing_Helper)
