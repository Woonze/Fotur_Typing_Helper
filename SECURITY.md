# Безопасность

## Поддерживаемые версии

Исправления безопасности выпускаются для последней опубликованной версии Fotur Typing Helper.

## Как сообщить об уязвимости

Не публикуйте детали уязвимости в открытом Issue до исправления. Используйте **Security → Report a vulnerability** в репозитории GitHub. Укажите версию Windows и Fotur, шаги воспроизведения, ожидаемое и фактическое поведение. Не прикладывайте настоящий пользовательский текст, аудио, API-ключи или другие секреты.

## Модель приватности

Основная автокоррекция, распознавание и перевод речи работают локально. Содержимое ввода не пишется в диагностический журнал; временный WAV удаляется после обработки. Сеть используется для первой загрузки модели. Будущие облачные провайдеры будут только опциональными и с ключом пользователя.
