# Каталог функций

Это живое описание версии 0.2.0. Текущий статус и известные ограничения находятся в [BACKLOG.md](BACKLOG.md).

## 1. Автоматическое исправление раскладки

Приложение перехватывает только необходимые события клавиатуры и сравнивает введённые клавиши с русской и английской раскладками. Оно оценивает не только текущее слово, но и предыдущий непрерывный контекст до 24 слов. Если вся фраза набрана не в той раскладке, удаляется неверный фрагмент, вставляется правильный и переключается раскладка активного приложения.

- **Надёжный режим** исправляет слово или фразу на пробеле, Enter либо знаке препинания. Это основной рекомендуемый режим.
- **Раннее исправление — экспериментальная функция.** Она может заменить слово уже после четвёртой буквы, но только при повышенной уверенности не ниже 90%. Пометка «экспериментально» означает, что режим полезен для скорости, но на редких именах, сленге и коротких словах возможна лишняя замена; его можно отключить.
- Кэш активного окна действует 200 мс, а потенциально зависающие Win32-запросы ограничены 30 мс, поэтому защита password-полей не должна задерживать печать.
- Клавиши, внедрённые самим Fotur, помечаются собственной сигнатурой и не анализируются повторно. Ввод от экранных клавиатур и средств доступности при этом остаётся доступен для обработки.
- Сразу после замены `Ctrl+Z` или назначенная пользователем клавиша отмены возвращает исходный текст.
- Пользовательский словарь повышает уверенность для имён, брендов и терминов.
- Список исключений полностью отключает анализ в выбранных процессах.

### Приватность и защищённые поля

Текст не сохраняется в журнал и не отправляется по сети. Обработка отключается в распознанных нативных password-полях, системных защищённых окнах, менеджерах паролей, банковских программах и приложениях из списка исключений. Некоторые браузеры рисуют собственные поля, поэтому абсолютную защиту одним Windows API гарантировать нельзя; расширенная UI Automation-проверка остаётся в backlog.

## 2. Голосовая диктовка

Диктовка запускается глобальной горячей клавишей в одном из двух режимов:

- удерживать сочетание, пока пользователь говорит;
- нажать один раз для старта и повторно для остановки.

Перед записью запоминается активное окно, показывается компактный микрофонный overlay, затем фокус возвращается в исходное окно и распознанный текст вставляется туда. Аудио обрабатывается локальной Whisper-совместимой моделью. Windows x64 runtime поставляется обычными native DLL рядом с приложением, потому что Whisper не может надёжно найти их внутри single-file EXE. Сборка останавливается, если `runtimes/win-x64/whisper.dll` отсутствует. Отдельные smoke-тесты проверяют загрузку runtime и полный цикл микрофон → WAV → модель → распознавание.

Поддерживаются русский, английский и автоматическое определение языка. Пунктуацию и регистр формирует модель. Голосовые команды «новая строка», «точка», «запятая», «удали последнее слово» можно отключить. Качество англицизмов, сленга и резких переходов RU↔EN зависит от размера модели: более крупная модель точнее, но требует больше памяти и времени.

## 3. Горячие клавиши

Сочетания диктовки и отмены исправления меняются в настройках: нужно сфокусировать поле и нажать желаемую комбинацию. При сохранении выполняется проверка дубликатов, системных сочетаний `Alt+F4`, `Ctrl+Shift+Escape`, `Ctrl+Alt+Delete`, обязательного модификатора и поддерживаемой клавиши. Некорректная комбинация не применяется, а интерфейс показывает причину.

## 4. Интерфейс, трей и жизненный цикл

- Профессиональная тёмная тема Fotur, единая навигация и статус локальной обработки.
- У каждой сложной настройки есть короткое описание, у длинных пояснений — кнопка `i` с подсказкой при наведении.
- Закрытие и сворачивание скрывают окно в системный трей; процесс продолжает работу.
- Меню трея открывает окно, включает функции, показывает модель и завершает программу по явной команде «Выход».
- Автозапуск хранится в пользовательском разделе Windows и не требует прав администратора.

## 5. Настройки и данные

Настраиваются общие переключатели, раннее исправление, порог уверенности, горячие клавиши, режим записи, локальная модель, голосовые команды, словарь, исключённые приложения, автозапуск и уведомления. Настройки и агрегированная статистика хранятся локально. Содержимое напечатанного и продиктованного текста не сохраняется.

Диагностический журнал содержит только технические ошибки и находится в `%LOCALAPPDATA%\Fotur\TypingHelper\logs`; текст пользователя туда не попадает.

## 6. Установка и обновления

Inno Setup-установщик поддерживает русский и английский язык, выбор папки, ярлык, автозапуск и удаление. Portable-версия не требует установки. Автоматическая проверка GitHub Releases, цифровая подпись и автообновление запланированы, но в 0.2.0 ещё не включены.
