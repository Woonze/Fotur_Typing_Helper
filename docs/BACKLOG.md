# Backlog и матрица готовности

Статусы: `[x]` реализовано и автоматически проверено, `[~]` реализовано, но нужна ручная проверка на целевых устройствах/приложениях, `[ ]` запланировано.

## Версия 1.0.0

- [x] Версия продукта, installer и документация синхронизированы на 1.0.0.
- [x] Реальный Chrome-стенд расширен до 150 RU/EN-фраз разной длины.
- [x] Перед каждой фразой стенд проверяет противоположную целевому языку раскладку; ввод идёт с задержкой 35 мс на клавишу.
- [x] Контекстная оценка выполняется по словам, исправляет частично смешанные фразы, сохраняет регистр и защищает бренд Fotur.
- [x] Диктовка обозначается мягким фирменным свечением от краёв к центру на всех мониторах; глубина эффекта ограничена 5% экрана.
- [x] Выбор микрофона, живой уровень, VAD, мягкое шумоподавление, длинная запись, prompt личного словаря, пунктуация и фильтр слов-паразитов.
- [x] Small — новая рекомендуемая модель; в UI указаны размеры и примерное потребление RAM.
- [x] Автообновление из стабильных GitHub Releases при запуске с проверкой SHA-256 и выключателем в настройках.
- [x] Расширенный экран «О программе»: создатель Кирилл/Woonze, версия, дата 23.07.2026, платформа, лицензия и репозиторий.
- [x] Исходный код разделён на переносимые контракты, Windows- и macOS-адаптеры.
- [x] GitHub Actions собирает macOS Apple Silicon на ARM runner и Intel на Intel runner, проверяет `.app`, ad-hoc codesign и нативный Whisper runtime, создаёт ZIP/DMG.
- [~] macOS: CGEvent автокоррекция, переназначаемая диктовка/отмена (включая Cmd, цифры и F1–F12), OpenAL microphone, CoreML/CPU Whisper, LaunchAgent и tray реализованы. Раздельно проверяются Input Monitoring и Accessibility/PostEvent; несостоявшаяся вставка больше не попадает в статистику. Повторная ручная проверка нового DMG должна пройти на личных Intel/M-series Mac.
- [~] DMG имеет ad-hoc подпись. Для бесшовного Gatekeeper и автообновления нужны платный Apple Developer ID и notarization; без них пользователь подтверждает первый запуск вручную.

## Диктовка и модели

- [x] Tiny/Base/Small/Medium, размеры/RAM и выбор в UI.
- [x] Выбор устройства и индикатор уровня.
- [x] VAD, noise gate и автоматическая сегментация длинной записи Whisper.
- [x] Prompt из личного словаря без сохранения диктовки.
- [~] Code-switching RU↔EN и англицизмы поддерживаются multilingual Whisper; качество Small/Medium требуется проверять на живом наборе голосов и акцентов.
- [x] Локальная постобработка пунктуации, регистра, пробелов и слов-паразитов.
- [x] Обычная транскрипция и локальный перевод на английский разделены.
- [ ] Стабильный потоковый предварительный текст во время записи. Нужна отдельная архитектура инкрементальных гипотез, чтобы не мерцать и не портить caret.
- [ ] Перевод на целевые языки, кроме английского. Встроенный Whisper этого не делает; требуется отдельная локальная translation-модель.

## Облачные BYOK-провайдеры

Локальный режим всегда остаётся полностью рабочим. Ключ должен принадлежать пользователю и храниться в защищённом хранилище ОС.

- [ ] Gemini Developer API / Google AI Studio — после проверки актуального бесплатного лимита и пригодности именно для streaming speech.
- [ ] Google Cloud Speech-to-Text BYOK.
- [ ] OpenAI Audio API BYOK.
- [ ] Azure Speech, Deepgram и другие после сравнительного RU/EN-теста.
- [ ] Единый интерфейс провайдеров, проверка ключа, прозрачная оценка стоимости и fallback на локальную модель.
