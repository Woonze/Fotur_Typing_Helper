# Установка Fotur Typing Helper 1.1.0 на macOS

## 1. Выберите сборку

- Mac с процессором Apple M1/M2/M3/M4/M5: `FoturTypingHelper-1.1.0-macos-arm64.dmg`.
- Mac с процессором Intel: `FoturTypingHelper-1.1.0-macos-x64.dmg`.

Тип процессора указан в меню Apple → «Об этом Mac».

## 2. Установите приложение

1. Откройте скачанный DMG.
2. Перетащите `Fotur Typing Helper` на ярлык `Applications`.
3. Извлеките DMG.
4. Откройте папку «Программы» и запускайте Fotur только оттуда. Не запускайте копию из Downloads, ZIP или открытого DMG: macOS привязывает разрешения к конкретному приложению.

## 3. Первый запуск

Пакет пока имеет ad-hoc подпись и не нотарифицирован Apple. Если обычный запуск заблокирован:

1. Откройте «Системные настройки → Конфиденциальность и безопасность».
2. Найдите сообщение о заблокированном `Fotur Typing Helper`.
3. Нажмите «Всё равно открыть» и подтвердите запуск.

Альтернативный штатный способ — Control-клик по приложению в папке «Программы», затем «Открыть».

## 4. Выдайте три разрешения

Откройте «Системные настройки → Конфиденциальность и безопасность» и включите для `Fotur Typing Helper`:

1. **Микрофон** — запись диктовки.
2. **Мониторинг ввода** — глобальные хоткеи и определение раскладки.
3. **Универсальный доступ** — удаление ошибочного текста и вставка исправленного текста.

Это независимые разрешения. Если одного нет, часть функций может показываться активной, но не сможет воздействовать на активное поле.

После изменения прав выберите в меню/трее Fotur «Выход», убедитесь, что приложение завершилось, и снова откройте его из «Программ».

## 5. Быстрая проверка

1. Откройте TextEdit и создайте обычный текстовый документ.
2. Проверьте автокоррекцию, набрав русскую фразу в английской раскладке.
3. Зажмите настроенную клавишу диктовки, произнесите короткую фразу и отпустите её.
4. Убедитесь, что при удержании буквенной горячей клавиши сама буква не печатается.

## Если хоткей или замена не работают

1. Полностью закройте Fotur.
2. В разделах «Мониторинг ввода» и «Универсальный доступ» удалите старую запись Fotur кнопкой `−`.
3. Добавьте именно `/Applications/Fotur Typing Helper.app` и включите переключатель.
4. Снова запустите приложение из «Программ».
5. Если macOS просит выйти и открыть приложение заново — обязательно сделайте это.

Лог приложения находится в `~/Library/Application Support/Fotur/TypingHelper/logs/fotur-typing-helper.log`.

Если проблема сохраняется, приложите к issue версию macOS, тип процессора, выбранный хоткей, название проверяемого приложения и последние строки лога. Не публикуйте личный продиктованный или набранный текст.
