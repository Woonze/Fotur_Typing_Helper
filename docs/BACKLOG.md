- [x] GitHub Actions workflow и адрес репозитория.

Каждая строка должна пройти ручной тест автокоррекции, отмены и диктовки.

|---|---:|---:|---|

| Chrome | [x] | [~] | автокоррекция проверена реальным SendInput на трёх фразах; диктовке нужен тест вставки речи в поле |
| Edge | [ ] | [ ] | password и contenteditable |
| Telegram Desktop | [ ] | [ ] | emoji и редактирование сообщений |

| VS Code | [ ] | [ ] | редактор Monaco |
