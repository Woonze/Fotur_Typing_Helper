# Fotur Typing Helper

Open-source помощник ввода для Windows 10/11 и macOS 12+. Версия 1.0.0 автоматически исправляет русско-английскую раскладку целыми фразами, запускает локальную диктовку по глобальной горячей клавише и остаётся доступной из системного трея.

## Возможности

- Исправление EN↔RU с сохранением регистра, контекста до 24 слов и смешанных фрагментов вроде `Fotur помогает`.
- Локальная диктовка Whisper: русский, английский, автоматическое определение и отдельный перевод речи на английский.
- Модели Tiny, Base, Small и Medium; по умолчанию Small.
- Выбор микрофона, живой индикатор, VAD, шумовой порог, словарный prompt, голосовая пунктуация и фильтр слов-паразитов.
- Фирменная бирюзово-розовая рамка по краям всех экранов во время записи.
- Переназначаемые горячие клавиши, проверка конфликтов, отмена исправления, исключения приложений.
- Автозапуск, работа в трее и автоматическое обновление из GitHub Releases с проверкой SHA-256.
- Windows x64 installer/portable и отдельные macOS DMG/ZIP для Apple Silicon и Intel.

Все распознавание выполняется локально. Интернет требуется для первой загрузки модели и проверки обновлений; текст и аудио не отправляются в облако. Временный WAV удаляется после распознавания.

## Загрузка и первый запуск

Готовые файлы публикуются в [GitHub Releases](https://github.com/Woonze/Fotur_Typing_Helper/releases). На macOS выберите `macos-arm64` для M1/M2/M3/M4/M5 либо `macos-x64` для Intel. При первом запуске macOS попросит доступ к микрофону, Accessibility («Универсальный доступ») и Input Monitoring («Мониторинг ввода»). Эти права независимы: первое разрешает Fotur заменять и вставлять текст, второе — видеть глобальные горячие клавиши. После их изменения полностью закройте Fotur через трей и запустите снова.

Сборки 1.0.0 пока имеют ad-hoc подпись: без Apple Developer ID приложение не может быть нотарифицировано Apple, поэтому Gatekeeper может потребовать подтверждение через «Системные настройки → Конфиденциальность и безопасность → Всё равно открыть». Это открытое ограничение, а не скрытая ошибка установщика.
